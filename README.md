# TiltMonitor — монитор углов наклона (акселерометр)

Программа для отображения углов наклона, получаемых с акселерометра через последовательный порт (COM). Данные отображаются на 3D‑модели (или тестовом кубе), а также на графике изменения углов.

![Иконка](icon.ico)

## Возможности

- Подключение к любому COM‑порту (скорость, окончание строки настраиваются).
- Периодический опрос датчика (команда `READ`).
- Отображение трёх углов (X, Y, Z) в реальном времени.
- Вращение 3D‑модели в соответствии с сопоставлением осей датчика и осей модели.
- Настройка сопоставления осей, инверсии, смещения начала отсчёта, отключения ненужных осей.
- График изменения углов (с возможностью показа/скрытия каждой оси и ограничения числа точек).
- Лог событий.
- 3D‑сцена с осями координат и сеткой (пол).

## Требования

- **Python 3.8+**
- Установленные библиотеки (см. список ниже):
  - `pyqt5`
  - `pyqtgraph`
  - `numpy`
  - `pyserial`
  - `PyOpenGL`
  - `pywavefront`

## Установка и запуск из исходников

1. Склонируйте или скачайте папку с проектом.
2. Установите зависимости:
   ```bash
   pip install pyqt5 pyqtgraph numpy pyserial PyOpenGL pywavefront
   ```
3. Запустите программу:
   ```bash
   python digital_tilt.py
   ```

## Сборка исполняемого файла (.exe)

Для создания автономного EXE‑файла используется скрипт `build_exe.py`. Он упаковывает программу вместе с 3D‑моделью и иконкой с помощью **PyInstaller**.

### Подготовка

- Убедитесь, что в папке есть файлы:  
  `digital_tilt.py`, `Tilt_Switch_Digital.obj`, `Tilt_Switch_Digital.mtl`, `icon.ico` (иконка для окна и файла).
- Установите PyInstaller:
  ```bash
  pip install pyinstaller
  ```

### Сборка

Выполните в командной строке (в папке проекта):
```bash
python build_exe.py
```
После завершения готовый `TiltMonitor.exe` появится в папке `dist/`.  
Туда же автоматически скопируются файлы модели (`.obj`, `.mtl`), которые должны лежать рядом с EXE для загрузки 3D‑модели.

**Примечание по размеру:**  
Если в вашей глобальной среде Python установлены тяжёлые библиотеки (PyTorch, TensorFlow, Pandas и т.д.), они могут случайно попасть в сборку. Чтобы этого избежать, используйте **чистое виртуальное окружение**:

```bash
python -m venv venv
venv\Scripts\activate
pip install pyqt5 pyqtgraph numpy pyserial PyOpenGL pywavefront pyinstaller
python build_exe.py
```

## Настройка подключения

1. Выберите **COM‑порт** из выпадающего списка (кнопка «Обновить»).
2. Установите **скорость** (бод) – обычно 115200.
3. **Окончание строки** – по умолчанию `\n (LF)`. Выберите то, что используется вашим устройством.
4. Нажмите **«Подключиться»**.
5. После подключения нажмите **«Старт опроса»** – программа начнёт отправлять команду `READ` через заданный интервал и отображать полученные углы.

## Управление отображением

### Сопоставление осей

В группе **«Сопоставление осей и начальное положение»** можно для каждой оси модели выбрать, какой канал датчика (X, Y или Z) будет её вращать, и при необходимости инвертировать направление.

- **Отключить ось** – фиксирует вращение модели по этой оси в положении, заданном смещением (см. ниже).
- **Смещение** – добавляет постоянный угол к соответствующей оси модели (работает даже при отключённой оси). Полезно для коррекции начального положения.

### График

- Выберите, какие оси показывать на графике (X – крен, Y – тангаж, Z – рыскание).
- Опция **«Отображать только последние 100 точек»** ограничивает длину графика для повышения производительности.

### 3D‑сцена

- Кнопка **«Скрыть 3D сцену»** / **«Показать 3D сцену»** – позволяет освободить место на экране.
- На сцене отображаются:
  - Оси координат (X – красный, Y – зелёный, Z – синий).
  - Сетка на плоскости XZ (пол).

## Формат данных от датчика

Устройство должно выдавать строки вида:

```
X: -0.839  Y: -13.496  Z: 71.834
```

Пробелы и точность могут быть произвольными. Программа автоматически извлекает числа и ограничивает их диапазоном **от -90 до +90** (акселерометр).

Команда опроса: **`READ`** (добавляется выбранное окончание строки).

## Примечания

- При отсутствии файлов 3D‑модели (`Tilt_Switch_Digital.obj`/`.mtl`) используется простой разноцветный куб.
- Файлы модели должны находиться в той же папке, что и исполняемый файл.
- Иконка окна (в панели задач) отображается только если файл `icon.ico` добавлен в сборку и код `digital_tilt.py` устанавливает иконку через `self.setWindowIcon()`
